# Natural Birth - Smart Doctor Directory

Discover top-rated gynecologists and obstetricians tailored to your preferences, birth plans, and cultural needs.

## 🛠 Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Backend**: FastAPI (Python) on Vercel Serverless
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Local Environment**:
   ```bash
   npm run vercel
   ```
   *Note: This uses the Vercel CLI to handle both Next.js and the Python API simultaneously.*

3. **Admin Dashboard**:
   Access the enriched data management at `/admin`.

## 📂 Features
- **Fuzzy Search**: Find doctors even with slight typos.
- **Natural Language Parsing**: Detects location and preference intents automatically.
- **Multilingual Support**: English and Bengali.
- **Dark Mode**: High-contrast premium dark theme.
- **Data Enrichment**: Automated web scraping from WordPress and Normal Delivery BD.
